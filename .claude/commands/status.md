TASK.md를 읽고 현재 프로젝트 상태를 요약해줘.

## 출력 형식

```
## 프로젝트 상태

### 진행률
[████████░░] 80% (4/5 완료)

### IN_PROGRESS
- T3: 티켓제목
  - AC 진행: 2/3 완료
  - 예상: 30분

### QUEUE (대기: n개)
1. T4: 티켓제목
2. T5: 티켓제목

### DONE (완료: n개)
- T1: 티켓제목 (01-15)
- T2: 티켓제목 (01-15)

### BLOCKED (막힘: n개)
- T6: 티켓제목
  - 원인: ...
  - 필요: ...

### 권장 액션
- (다음에 할 일 제안)
```

## 추가 확인
- 마지막 커밋 시간
- 테스트 상태 (최근 실행 결과)
